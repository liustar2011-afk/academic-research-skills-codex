# 汇入主线建议

## 汇入摘要

B-007 已形成电力行业政策依据补充，建议先汇入 `04_literature_policy/政策依据表.md`，用于支撑电力行业场景必要性、工程落地化、安全稳定约束和生态扩散化。

## 建议直接吸收的内容

1. 新型电力系统建设带来电力科技成果转化需求。
2. 能源科技创新规划说明成果转化对象包括装备、系统、软件、数字化智能化和绿色低碳技术。
3. 电力成果转化必须考虑安全稳定、工程验证、验收评价和责任追溯。
4. 首台套示范、设备更新和标准化政策支撑成果从研发到示范再到推广。

## 建议谨慎吸收的内容

1. 能源装备示范和设备更新政策可支撑工程应用场景，但不能替代科技成果转化政策。
2. 电力行业政策支撑“场景必要性”，不直接证明中电联运营能力。
3. 具体技术领域优先级还需结合访谈或案例。

## 不建议吸收的内容

1. 不建议把本研究写成新型电力系统总体建设研究。
2. 不建议深入展开电力系统稳定技术细节。
3. 不建议把设备更新需求直接等同于全部科技成果转化需求。

## 主线更新顺序

1. 更新 `04_literature_policy/政策依据表.md`。
2. 更新 `09_evidence/source_manifest.md` 和 `09_evidence/evidence_pool.md`。
