# 汇入主线建议

## 汇入摘要

B-002 已形成科技成果转化政策链条：成果转化法及配套规定、转移转化行动方案、国家技术转移体系、职务成果赋权、成果评价机制、专利转化运用和最新工作趋势。建议先汇入 `04_literature_policy/政策依据表.md`，再用于修订论文提纲和理论框架。

## 建议直接吸收的内容

1. “科技成果转化不是单点撮合，而是跨主体、跨要素、跨环节的系统工程。”
2. “中电联可被定位为行业级技术转移网络的枢纽型组织。”
3. “运营模式应覆盖成果资产管理、价值评价、权属核验、供需匹配、技术经纪、知识产权运营、科技金融、中试熟化、应用推广和生态扩散。”
4. “可信数据空间应被作为成果转化场景运营的基础设施能力，用于支撑可信发布、受控共享、评价留痕、交易追溯和履约验证。”

## 建议谨慎吸收的内容

1. 职务科技成果赋权政策主要面向试点单位，应谨慎外推至所有电力企业。
2. 专利转化政策可支撑知识产权运营，但科技成果不等于专利，不能把成果转化窄化为专利转化。
3. 科技金融政策启示较明确，但具体金融产品、合规边界和风险分担机制还需另设支线研究。

## 不建议吸收的内容

1. 不建议直接写“中电联已具备国家技术转移平台功能”。
2. 不建议直接写“可信数据空间可替代技术市场”。
3. 不建议直接写“所有电力成果都适合作价入股或专利产业化”。
