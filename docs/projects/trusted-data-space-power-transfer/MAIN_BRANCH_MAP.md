## 关联规则

1. 支线必须对应至少一个主线任务。
2. 支线必须说明主线缺口，不能为了“多研究”而研究。
3. 支线输出默认不直接覆盖主线成果，必须先形成汇入建议。
4. 汇入主线前必须检查 `DEPENDENCY_MAP.md`。
5. 汇入后必须更新 `PROJECT_INDEX.md`、`TASK_BOARD.md`、`ARTIFACTS.md`、`PROCESS_ARTIFACTS.md`、`CHANGELOG.md`。
