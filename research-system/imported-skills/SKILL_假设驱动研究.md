# SKILL｜假设驱动研究

## 适用场景

研究问题复杂、材料很多、方向不清，需要快速建立研究路径。

## 输出

```text
初始假设
假设树
议题优先级
证据需求
验证方式
修正记录
```

## 模板

```markdown
# 假设日志

| 编号 | 假设 | 支撑证据 | 反驳证据 | 状态 | 下一步验证 |
|---|---|---|---|---|---|
| H1 |  |  |  | 待验证/成立/修正/放弃 |  |
```

## 质量标准

- 每条假设必须可验证。
- 每条假设必须对应证据需求。
- 被反驳的假设应保留，不应删除。
- 关键判断应来自验证后的假设。
