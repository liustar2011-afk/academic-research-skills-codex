# SKILL｜研究成果资产

## 适用场景

需要把零散研究转为可复用母本，后续派生 PPT、Word、机器图和汇报材料。

## 必须包含

```text
研究对象
决策问题
研究边界
关键判断
证据映射
价值池
主体责任
战略选项
运营模式
实施路线
风险边界
可转化成果
```

## 输出模板

```markdown
# 研究成果资产

## 1. 研究对象与边界

## 2. 决策问题

## 3. 关键判断

## 4. 证据映射

## 5. 价值池与主体关系

## 6. 战略选项与推荐推进方式

## 7. 运营模式与实施路径

## 8. 风险边界与待核验事项

## 9. 可转化成果
```

## 闸门

研究成果资产不足时，不生成 PPT / Word。
