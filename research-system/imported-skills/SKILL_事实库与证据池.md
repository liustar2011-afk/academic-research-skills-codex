# SKILL｜事实库与证据池

## 适用场景

需要从政策、报告、PPT、Word、访谈纪要、数据表中提取证据，支撑研究判断。

## 输出

1. source_manifest。
2. fact_cards。
3. evidence_pool。
4. evidence_gap。
5. 待核验清单。

## 证据卡模板

```markdown
# Fact Card

编号：
事实：
来源：
类型：政策/业务/能力/市场/风险
支撑的判断：
证据强度：A/B/C/D
是否可进入正式材料：是/否
待核验：
```

## 禁止

- 只摘录原文。
- 只列文件名。
- 无证据强度分级。
- 删除反例或不利证据。
