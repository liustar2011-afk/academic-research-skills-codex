# SKILL｜研究成果转 PPT

## 适用场景

已形成 research_asset，需要转为汇报型 PPT、项目介绍 PPT、高层交流 PPT 或场景设计 PPT。

## 输入

- research_asset.md
- decision_memo.md
- 交付对象
- 页数要求
- 参考风格

## 输出

每页包含：

```text
页面职能
核心判断
研究资产来源
构图主链
页面构图
框内文字终稿
箭头与关系
组件清单
自检标准
```

## 规则

- 每页只承担一个职能。
- 页面主链必须显性化。
- 文字来自研究资产，不从原始材料直接摘抄。
- 不得把研究过程、自检说明、页面职能写入页面正文。
- 不得降低信息密度。
