# 领域适配器

领域适配器用于在研究成果转化阶段校准政策口径、行业角色、表达结构和自检标准。

## 使用规则

1. 领域适配器不替代事实核验。
2. 领域适配器只在已有研究任务书、证据池和研究成果资产后启用。
3. 每个项目应在 `10_domain_adapter/domain_selection.md` 中记录适配器选择。
