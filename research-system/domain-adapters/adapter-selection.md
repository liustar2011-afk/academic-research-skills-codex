# 领域适配器选择

## 适用场景

当任务涉及具体领域研究，尤其是电力、能源、数据基础设施、可信数据空间、AI 数字化、行业公共服务、高层合作或正式材料转化时使用。

## 输入要求

- 研究对象。
- 交付场景。
- 服务对象。
- 已有材料。
- 期望交付物。

## 输出

- 主适配器。
- 辅助适配器。
- 选择理由。
- 证据清单。
- 质量闸门。
- 交付转化建议。

## 处理流程

```text
识别研究对象和交付场景
→ 选择主适配器
→ 判断辅助适配器
→ 生成 domain_selection.md
→ 基于适配器补齐领域证据清单
→ 将领域判断汇入 research_asset.md
```
