# 中电联行业组织型材料转化

## 适用对象

中电联正式报告、工作方案、请示汇报、PPT 脚本、行业统计分析、标准规则材料、会员服务方案、可靠性、信用、国际合作、科技成果转化、数据服务场景设计。

## 使用边界

本适配器不作为研究起点，不直接生成正式报告、PPT 脚本或工作方案。

调用前必须先完成：

- 研究任务书。
- 决策问题。
- 议题树/假设树。
- 证据池。
- research_asset.md。

## 核心定位

中电联相关材料应突出：

- 行业组织者。
- 规则支撑者。
- 公共服务者。
- 场景运营组织者。

默认表述：

> 依托行业组织、标准规则、统计监测、专家资源、会员联系、专业服务和行业数据基础，服务政府决策、支撑行业治理、服务会员企业、促进产业协同，形成兼具公共服务属性和场景运营能力的行业组织型平台。

## 输出结构

### 正式报告

一、背景形势与工作依据  
二、行业需求与现实基础  
三、中电联定位与能力基础  
四、重点任务与场景设计  
五、组织实施与运营机制  
六、风险边界与保障措施  
七、下一步工作建议

### 场景设计

场景名称 → 服务对象 → 业务痛点 → 中电联基础 → 数据/规则/专家/会员资源 → 服务内容 → 交付成果 → 运营机制 → 合规边界 → 推广路径

## 中电联承接主线

政策要求 → 行业需求 → 中电联基础 → 场景切口 → 服务产品 → 试点验证 → 行业推广

任何材料都要写清：

- 政策从哪里来。
- 行业缺什么。
- 中电联有什么基础。
- 中电联能组织谁。
- 能形成什么服务。
- 如何试点验证。
- 如何复制推广。
- 如何守住边界。

## 自检清单

- 中电联行业组织定位是否准确。
- 中电联、数智公司、会员企业、专家机构和外部合作方角色是否清晰。
- 行业需求、资源基础和服务产品关系是否明确。
- 标准、统计、会员、专家、数据等基础支撑是否充分。
- 场景落地和运营机制是否具体。
- 是否存在越位、错位、自我拔高或商业营销化表达。
