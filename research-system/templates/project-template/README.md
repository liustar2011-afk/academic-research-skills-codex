# PROJECT_NAME

本目录用于隔离和沉淀一个独立研究项目的全部资料、成果和管理文件。

## 当前入口

- `PROJECT_INDEX.md`
- `CURRENT_STATE.md`

## 目录结构

| 目录 | 用途 |
|---|---|
| `00_project_brief/` | 项目说明、研究问题、阶段目标 |
| `01_source_materials/` | 本地材料索引、外部资料清单、资料使用边界 |
| `02_stage_outputs/` | 阶段性成果、材料诊断、阶段小结 |
| `03_outline/` | 开题报告提纲、论文提纲、章节结构 |
| `04_literature_policy/` | 政策依据表、文献综述笔记、参考文献库 |
| `05_models/` | 理论分析框架、运营模式、流程模型 |
| `06_drafts/` | 开题报告草稿、论文初稿、汇报稿 |
| `06_synthesis/` | 研究成果资产母本和跨成果综合材料 |
| `07_branch_research/` | 支线研究任务与成果 |
| `08_review_revision/` | 外部反馈、修改意见、修订记录 |
| `09_evidence/` | 来源清单、证据池、证据缺口、待核验事项 |
| `10_domain_adapter/` | 领域适配器选择和口径说明 |
| `11_versions/` | 阶段版本留存 |
| `99_archive/` | 过期材料和历史归档 |

## 通用管理文件

| 文件 | 用途 |
|---|---|
| `PROJECT_INDEX.md` | 项目控制台 |
| `TASK_BOARD.md` | 任务板 |
| `ARTIFACTS.md` | 成果物索引 |
| `PROCESS_ARTIFACTS.md` | 过程成果物索引 |
| `CURRENT_STATE.md` | 中断恢复状态 |
| `CHANGELOG.md` | 变更记录 |
| `DEPENDENCY_MAP.md` | 依赖关系 |
| `REVISION_QUEUE.md` | 修订队列 |
| `DECISIONS.md` | 决策记录 |
| `HYPOTHESIS_LOG.md` | 假设日志 |
| `review_log.md` | 复核记录 |
| `artifact_manifest.md` | 成果清单 |
