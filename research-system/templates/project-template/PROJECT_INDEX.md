## 中断后恢复提示

恢复项目时，按以下顺序读取：

1. `CURRENT_STATE.md`
2. `PROJECT_INDEX.md`
3. `CHANGELOG.md`
4. 本次要修改的具体文件

如果要修改上游成果，先查看：

1. `DEPENDENCY_MAP.md`
2. `REVISION_QUEUE.md`
3. `DECISIONS.md`

## 文件状态约定

| 状态 | 含义 |
|---|---|
| draft | 初稿，可大幅修改 |
| baseline | 当前阶段基准稿 |
| review | 待检查或待确认 |
| validated | 已确认，可作为下游依据 |
| integrated | 已汇入主线 |
| superseded | 已被新版本替代 |
| final | 定稿 |
