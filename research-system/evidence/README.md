# 事实库与证据池

本模块用于把政策、报告、PPT、Word、访谈纪要、数据表中的信息转化为可追溯证据。

## 标准输出

| 文件/目录 | 用途 |
|---|---|
| `source_manifest.md` | 来源清单 |
| `fact_cards/` | 单条事实卡 |
| `evidence_pool.md` | 证据池 |
| `evidence_gap.md` | 证据缺口 |
| `verification_todo.md` | 待核验清单 |

## 证据强度

| 等级 | 类型 |
|---|---|
| A | 法律、国家级政策、正式部委文件 |
| B | 行业标准、官方解读、权威报告 |
| C | 学术论文、行业白皮书、公开案例 |
| D | 项目材料、访谈、内部判断 |

## 使用规则

1. 不只摘录原文，必须说明支撑哪个判断。
2. 不只列文件名，必须标注证据强度和待核验事项。
3. 不删除反例或不利证据。
4. 证据池与 `CLAIM_EVIDENCE_MAP.md` 联动。
