# 公文排版参照规范 GB/T 9704-2012

## 适用场景

用于将 Word 文稿、正式报告、请示、通知、函、纪要、报送稿、内部流转稿等材料，整理为符合或参照 GB/T 9704-2012《党政机关公文格式》的正式公文版式。

本规范只处理版式、格式要素、层级样式和排版自检，不替代行文内容审定、发文程序审核、印章管理和密级管理。

## 页面规格

| 项目 | 规则 |
|---|---|
| 纸张 | A4，210mm × 297mm |
| 上白边 | 37mm ± 1mm |
| 左白边 / 订口 | 28mm ± 1mm |
| 版心 | 156mm × 225mm |
| Word 实操参考 | 上 3.7cm，左 2.8cm，右 2.6cm，下 3.5cm |

## 字体字号

| 要素 | 推荐规则 |
|---|---|
| 一般格式要素 | 3号仿宋体 |
| 标题 | 2号小标宋体，居中 |
| 第一层标题 | 3号黑体 |
| 第二层标题 | 3号楷体 |
| 第三、四层标题 | 3号仿宋体 |
| 版记 | 4号仿宋体 |
| 页码 | 4号半角宋体阿拉伯数字 |

## 禁止事项

- 不擅自生成密级、份号、发文字号、签发人、印章。
- 不把页面职能、写作意图、自检说明、执行器提示写入正文。
- 不因套格式删除正文中的有效业务信息。
- 不把研究成果直接压缩成空泛公文套话。
- 不使用与正文风格冲突的花哨字体、彩色强调、图标化装饰。
